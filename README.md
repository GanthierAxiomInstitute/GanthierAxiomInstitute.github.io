# Ganthier Academy(Ganthier Axiom Institute) Website

Upload all files and folders to the GitHub repository root.

Put every prepared image into `assets/images/` and keep the original file names, including `Logomark.png`, `LOGO2.jpg`, `poster-1.jpg`, etc.

This version includes inline CSS inside every HTML page, so the layout will still display correctly even if GitHub Pages has trouble reading the external CSS file.
